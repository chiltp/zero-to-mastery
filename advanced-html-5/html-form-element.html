HTML Form Elements

1. The <form> Element

The HTML <form> element is used to create an HTML form for user input. It is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

2. The <label> Element

The <label> element represents a caption for an item in a user interface.

  • The <label> element is used to provide a text description or label for a specific form input element. It helps users understand the purpose or expected input of the associated form field.

This linkage enables screen readers and other assistive technologies to understand the relationship between the label and the form element, enhancing accessibility for all users.

3. The <input> Element

The HTML <input> element is the most used form element. It can be displayed in many ways, depending on the type attribute.

4. The <select> Element

The select element defines a drop-down list.

  1. The option elements defines an option that can be selected. By default, the first item in the drop-down list is selected.
  2. To define a pre-selected option, add the selected attribute to the option.
  3. Use the size attribute to specify the number of visible values

  4. Use the multiple attribute to allow the user to select more than one value.
