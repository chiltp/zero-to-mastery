The Name Attribute for <input>

Notice that each input field must have a name attribute to be submitted. If the name attribute is omitted, the value of the input field will not be sent at all.

The "name" attribute serves two primary purposes:

  1. Grouping: By giving radio buttons the same "name" attribute, you specify that they belong to a common group. In this case, all the radio buttons with the "name" attribute set to "fav_language" are part of the same group.
  2. Selection Limitation: The "name" attribute ensures that only one radio button in the group can be selected at a time. When a user selects one option, it automatically deselects any other radio button with the same "name" attribute within the group.
