HTML Input Types

1. Text

The <input type="text"> defines a single-line input field for text input.



2. Password

The <input type="password"> defines a password field.


3. Submit

The <input type="submit"> defines a button for submitting form data to a form-handler. The form-handler is typically a server page with a script for processing input data.

Note: If you omit the submit button's value attribute, the button will get a default text.

4. Reset

The <input type="reset"> defines a reset button that will reset all form values to their default values.

Note: If you change the input values and then click the "Reset" button, the form-data will be reset to the default values.

5. Radio

The <input type="radio"> defines a radio button. Radio buttons let a user select ONE of a limited number of choices.

7. Email

The <input type="email"> is used for input fields that should contain an e-mail address.
