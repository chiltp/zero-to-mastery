Some tips from freeCodeCamp

1. Make dead links using the hash symbol

Using the hash symbol when you want to create placeholders for links that don't have an actual destination or are not yet implemented.

Click here

The href attribute is set to "#" to represent an empty URL or a link that doesn't lead anywhere. When the user clicks on the link, nothing will happen or change in the page because the link is effectively dead.
