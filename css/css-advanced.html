Responsive UI

Responsive web design makes your web page look good on all devices.

Responsive web design uses only HTML and CSS.

Responsive web design is not a program or a JavaScript.

Viewport

  • In CSS, the viewport refers to the visible area of a web page displayed within a browser window. It is essentially the area where the content of your webpage is rendered and displayed to the user.

  • The viewport is important for creating responsive web designs that adapt to different screen sizes and devices. By properly setting the viewport, you can control how your webpage is displayed and ensure that it looks good and functions well across various devices, such as desktops, laptops, tablets, and mobile phones.

  • To set the viewport, you can use the <meta> tag in the <head> section of your HTML document. Here's an example of how you can set the viewport:

    • <meta name="viewport" content="width=device-width, initial-scale=1.0">

  • Here are a few common viewport properties you can use within the content attribute:

    • width=device-width: Sets the width of the viewport to the width of the device screen. This ensures that the webpage adjusts its layout based on the device's screen size.

    • initial-scale=1.0: Sets the initial zoom level of the webpage. A value of 1.0 means no zoom, and the webpage will be displayed at its normal scale.

    • user-scalable=yes/no: Specifies whether the user can zoom in or out of the webpage. Setting it to "yes" allows zooming, while "no" disables zooming.

    • minimum-scale=0.5 and maximum-scale=2.0: Define the minimum and maximum zoom levels, restricting the user from zooming below or above the specified values.
